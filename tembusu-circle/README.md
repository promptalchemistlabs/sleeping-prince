# Tembusu Circle

A separate Gatsby demo business used by Kingdom of PAL to simulate an agent-operated terrarium community. All business pages and articles live in `content/` as Markdown.

```bash
npm install
npm run develop
npm run build
```

The orchestration UI treats Scribe artifacts as reviewable drafts. Approved website pages map to `content/pages/<slug>.md`; approved blog posts map to `content/blog/<slug>.md`.
